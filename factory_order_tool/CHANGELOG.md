# 更新日志

## [1.1.1] - 2026-03-02

### 修复
- 修复图纸版本号提取错误：原正则误匹配修订历史表头中的冒号，改为仅匹配标题栏"版本号 REV"字段
- 修复图纸文件查找失败：适配"J编号 YY编号"格式的文件名，不再要求文件名以YY开头
- 修正日期映射逻辑：计划开始时间改为下单当天，计划结束时间填入订单交货日期；交货日期早于今天时自动取今天（防止系统报错）

### 界面
- 图纸比对状态栏改为中文显示（匹配/不匹配/无图纸/无版本/无法提取/跳过）
- 预览表格新增"交货日期"列，"出货日期"更名为"开始日期"

## [1.1.0] - 2026-03-01

### 映射表适配
- 适配新版料号清单结构（5个sheet: 导线/带胶壳/0002/0003/0005）
- 每个sheet自动查找含「久益料号」的表头行，从下一行开始读数据
- 使用固定列位置读取（B=久益料号、C=客户料号、D=品名规格）
- 处理数值型客户料号（如sheet 0005中的整数）

### 输出简化
- 仅填充5个必填字段：产品编号、产品规格、数量(+5安全库存)、计划开始时间、工单分类(生产工单)
- 数量字段自动在采购数量基础上+5（安全库存考虑）
- 出货日期映射到「计划开始时间」列（原为计划结束时间）
- 其余14列留空

### 图纸版本比对（新功能）
- 新增图纸库选择和版本比对功能
- 版本号来源优先级：订单PDF交期回复列 → 映射表品名规格末尾
- 从图纸PDF中自动提取REV/版本号字段进行严格字符串比对
- 版本匹配的图纸自动复制到待打印文件夹
- 版本不匹配的红色高亮显示差异（本地版本 → 最新版本）
- 「一键全部打印」按钮（仅当全部匹配时启用）
- 「打开待打印文件夹」按钮
- 「我已完成最新图纸文件下载」按钮用于重新比对

### PDF解析
- 新增提取交期回复列（用于用户通过PDF编辑器填写的最新图纸版本号）

### 界面
- 预览表格精简为7列（移除空字段列）
- 新增图纸比对区域（图纸库选择、比对结果表格、操作按钮）
- 窗口尺寸调整为1150x900

### 打包
- 修复映射表被打包进_internal文件夹的bug（移除--add-data，改用post-build复制）

## [1.0.0] - 2026-02-27
首次发布

### 功能
- 解析生久科技采购单PDF，自动提取全部订单行（支持多页、6/7列自适应）
- 按产品定义映射表将客户料件编号(YY...)映射为工厂产品编号(J...)
- 输出符合「导入产品明细模板」19列格式的Excel文件
- GUI预览：绿色=映射成功，红色=未映射
- 一键打开映射表用Excel编辑，热重载无需重启
- 未映射编号自动汇总提醒
